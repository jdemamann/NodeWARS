# TentacleWars Visual Identity Pass

Date: 2026-03-12

## Scope

Dedicated TentacleWars render pass focused on:

- node identity
- lane organic feel
- packet presentation
- clash readability
- sandbox overlay tone
- mobile / desktop visual validation

## Main changes

- TentacleWars cells now render with softer circular membrane shells instead of inheriting the more angular NodeWARS shell language
- TentacleWars active lanes now use elongated packet droplets plus a faint peristaltic membrane pulse
- TentacleWars clash visuals now read less like a generic warning marker and more like a compressed biotic contact point
- sandbox hover/info overlays were reduced in weight so they interfere less with combat reading
- sandbox background gained a subtle atmospheric pass to separate the mode from the stable campaign look

## Outcome

- mode identity is now materially clearer at first glance
- TentacleWars no longer reads like a lightly recolored NodeWARS shell
- lane ownership and packet motion remain readable in low graphics
- the earlier oversized slice-adjacent visual bug is no longer present in the validation captures

## Residual gap

- very dense late-fight overlaps can still benefit from a future Wave B pass focused only on high-density corridor separation
